Project name: "Paradise Nursery"
Title: "e-plantShopping"
Online plant store where you can buy different kinds of plants.
Landing page:
It contains the projects name, and a short description about the project and what it does. Basically it is an online store for plants where a user can visit and buy different kinds of plants online.
The landing page links to the ProductList page with a Get started button.
Product List Page:
This Displays the plants available for sale in their different categories with each plant's name, decription, cost and an add to cart button enabling the user to select what they wish to buy. It also links back to the home/landing page by Clicking on the project name and also links to the cart Items page by cliking on the cart feature.
Cart Items page.
By clicking on the cart feature, you get to the cart items page which displays the current items in the cart. Each item on this page has a "-" button for reducing the quantity of that plant that a user wants to buy, a "+" button for increasing the quantity of that plant to buy, and a "delete" button to remove that plant from the cart.The total cost of each item is displayed on that item card and the over all total cost of all items in the cart is displayed at the top of the cart items card.
The continue to shopping button links back to the product List page for the user to continue adding items to cart. The cart items page also has links to the Product List page and the home/landing page on its navigation bar.
